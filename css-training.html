CSS Units and Lengths

Pixels

1px
100px
.box {
  width: 60px;
  height: 60px;
  background: deepskyblue;
}
.box

1px !== 1 pixel on the screen

Sarcasm:What! Why! That's Dumb!
Then all text on phones would have been tiny, they're wider than small monitors in pixels
We'd all be doing this (picture of user with magnifying glass over the screen)
1px = 1 pixel * ScreenDensity

Screen pixel density usually ranges from:
1x, 1.25x, 1.5x, 2x, or 4x

Perentage

1%
33.333%
150%
.parent {
  border-color: gray;
  border-style: dashed;
}

.box {
  width: 50%;
  background: deepskyblue;
}
.box

Perentage ... of what?

Usually it's parent element's width, but check documentation when it doesn't do what you expect.

Text Based Units

em

1em
1.25em
3em
.box {
  width: 2em;
  background: deepskyblue;
}
.box-2 {
  width: 2em;
  font-size: 2em;
  background: firebrick;
}
.box
.box-2

1em = Calculated font-size of current element

This may be confusing for some of you, because it has nothing to do with the width of an "em dash". Draw out calculation examples.
