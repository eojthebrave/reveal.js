The Landscape

Google Cardboard v1 & 2 will be Web VR's IE6

Web VR

Janus VR

MozVR

  • mozvr.com
  • Working on a WebVR API spec
  • Working on frameworks/libraries that can be used today

WebVR Boilerplate

  • Has everything you need to make a WebVR/3D experience in modern browsers
  • Very JS Heavy
  • Developer centric

A-Frame

  • Built on WebVR Boilerplate
  • Markup language that creates Web VR experiences
  • Designer Friendly(ish)
  • Can further be modified with JS
  • Project goal is to enable more creators
